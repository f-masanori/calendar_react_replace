# setup
## firebase

プロジェクトのルートディレクトリで以下のコマンドを実行し、`.firebase.env`が作成されたことを確認
```
$ make firebase.env
```

1. firebase project作成
2. firebase app作成
3. 認証に必要なキーを取得
4. `.firebase.env`のそれぞれの値に３で取得したキー等を入力
